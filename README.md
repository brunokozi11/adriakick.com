# adriakick.com
AdriaKick je portal posvećen hrvatskom i međunarodnom nogometu. Donosimo najnovije vijesti, transfere, analize utakmica, profile igrača, priče o mladim talentima i ekskluzivne sportske sadržaje. Pratite hrvatsku sportsku scenu i najvažnija događanja iz svijeta nogometa na jednom mjestu.
